# Task 00 - API Workspace Setup

Initial ASP.NET Core Web API workspace for Phase 02.

## Run the API

```bash
cd phase-02-web-api-basics/task-00-api-setup/ApiSetup
dotnet run
```

## Open Swagger

After the app starts, open:

- HTTPS: https://localhost:7146/swagger
- HTTP: http://localhost:5029/swagger

You should see the Swagger UI with the sample WeatherForecast endpoint.

## What is configured

- Controller-based Web API
- Swagger / OpenAPI in Development
- HTTPS redirection
