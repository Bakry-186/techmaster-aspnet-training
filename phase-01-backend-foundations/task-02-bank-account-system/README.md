# Task 02 - OOP Bank Account System

Console bank system with models, service layer, and menu UI.

## Run

```bash
cd phase-01-backend-foundations/task-02-bank-account-system/BankAccountSystem
dotnet run
```

## Features

- Create customer account
- Deposit / withdraw / transfer
- View account details and transaction history
- Balance changes only through account methods (encapsulation)

## Structure

```
BankAccountSystem/
  Models/
  Services/
  UI/
  Program.cs
```

## Demo checklist

- Create two accounts
- Deposit and withdraw
- Transfer between accounts
- Show one invalid case (e.g. insufficient balance)
