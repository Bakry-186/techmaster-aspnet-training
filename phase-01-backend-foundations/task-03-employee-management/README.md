# Task 03 - Employee Management

HR console app with seed data, search, filter, sort, and salary reports.

## Run

```bash
cd phase-01-backend-foundations/task-03-employee-management/EmployeeManagement
dotnet run
```

## Features

- Add / update / deactivate employees
- Search by ID or partial name
- Filter by department (active only)
- Sort by salary, hire date, or name
- Salary reports and department counts

Seed data includes 12 employees as required.
